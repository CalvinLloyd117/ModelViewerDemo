"Lazy Loading"

Despite .gltf and .glb files being considered the most efficient and widely supported for displaying 3D items on the web, some older devices might need some extra help to ensure a smooth experience.

Fortunately, Model-Viewer allows for "Lazy-Loading" which can help reduce the strain on slower/older hardware. Essentially, this technique replaces a 3D model with a picture which represents the model. Then the poster will be replaced with the 3D model when required, rather than having the model rendered right away. This will certainly help older devices when it comes to rendering on the web, especially when multiple models may be present on a page at a time.
